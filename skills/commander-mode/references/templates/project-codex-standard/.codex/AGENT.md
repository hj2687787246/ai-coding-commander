# 项目 `.codex` 工作手册

## 作用

- 定义当前项目的 commander 恢复顺序
- 约束当前项目的任务推进、验收和归档
- 说明项目内文档分工

## 恢复顺序

1. `.codex/docs/恢复入口.md`
2. `.codex/docs/当前状态.md`
3. `.codex/docs/当前任务.md`
4. `.codex/docs/协作偏好.md`

## 文档分工

- `当前状态.md`：当前项目整体状态
- `当前任务.md`：本轮正在推进的任务
- `验收记录.md`：已正式完成的事项
- `归档索引.md`：已过期或已降级内容
- `协作偏好.md`：当前项目用户的稳定协作偏好
- `周总结.md`：周级总结与轻量记忆审计
